## 功能设计
建议分成 4 个页面。

# 1. TCP 通信调试页面
功能：

输入 IP 和端口
连接 / 断开设备
发送十六进制指令或字符串指令
接收设备返回数据
显示通信日志
连接失败、超时、断开重连提示


# 2. 串口通信调试页面
功能：

选择 COM 口
设置波特率、数据位、停止位、校验位
打开 / 关闭串口
发送指令
接收返回数据
支持 HEX / ASCII 显示


# 3. Modbus TCP 调试页面
功能：

连接 Modbus TCP Server
读取保持寄存器
写入单个寄存器
显示寄存器地址和值
异常码提示

你可以用现成库，比如：

NModbus



4. 通信日志与配置页面
功能：

自动记录发送时间、接收时间、协议类型、发送内容、接收内容、状态
日志导出 Excel
保存连接配置
读取历史配置
